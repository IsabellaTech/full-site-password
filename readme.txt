Displays a password screen for the entire website until the correct password is entered.

== Description ==
Full Site Password is a WordPress plugin that provides site-wide protection by displaying a password screen across the entire website until the correct password is entered. This ensures that only authorized users can access the website's content.

== Installation ==
1. Upload the entire `full-site-password` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
Q: How do I set the password for the entire site?
A: To activate the Full Site Password feature, navigate to the 'Site Password' section in the WordPress admin area after installing and activating the plugin. Check the box labeled 'Activate Full Site Password' and save your settings.

Q: How can I customize the appearance of the password screen?
A: You can customize the appearance of the password screen by adjusting various settings in the plugin's options. These settings include background color, text color, top logo size, main logo size, and more.

Q: Can I display a custom title and description on the password screen?
A: Yes, you can. In the plugin's options, you can enter a custom title and description that will be displayed on the password screen.

Q: What happens if a user enters the wrong password?
A: If a user enters the wrong password, they will see an error message indicating that the password is incorrect. You can customize this error message in the plugin's options.

Q: How do I add custom logos and background images to the password screen?
A: To add custom logos and background images to the password screen, upload the desired images using the provided fields in the plugin's options. You can then adjust the image sizes and positions as needed.
